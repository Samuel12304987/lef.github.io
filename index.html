Over mezelf

Ik heet Samuel van Benschop. Ik zit in A5B op het vlietland college. Ik woon in Voorschoten en doe veel aan sport. Ik werk in een supermarkt in het centrum van voorschoten. Ik woon met mijn vader, moeder, broertje en twee huisdieren in een huis. Ik weet nog niet zo goed wat ik wil gaan doen als ik klaar ben met de middelbare school, maar ik denk iets met economie of bedrijven. Hierbij is informatica erg handig, vandaar dat ik dat gekozen heb. Hier ga ik verder op in in een ander stukje.

Mijn website

Deze website heb ik gemaakt voor het vak Informatica. De website is opgebouwd uit verschillende onderdelen die ik volledig zelf heb gecodeerd, waarbij ik gebruik heb gemaakt van HTML en CSS.De website is ontworpen om zowel functioneel als gebruiksvriendelijk te zijn. Ik heb specifiek aandacht besteed aan het design van de pagina, zodat de website er goed uitziet. Mijn website functioneert op zowel desktop- als mobiele apparaten. Daarnaast bevat de website interactieve elementen, zoals een dynamische navigatiebalk en een interactief spelletje dat de bezoeker van mijn website kan spelen.

Ook heb ik ervoor gezorgd dat de website eenvoudig navigeerbaar is, zodat bezoekers snel kunnen vinden wat ze zoeken. De dynamische navigatiebalk past zich automatisch aan wanneer je door de pagina scrollt, wat zorgt voor een soepel gebruik. Dit maakt het dat de website er niet alleen goed uit ziet, maar ook praktisch in gebruik.

Informatica

Ik heb informatica gekozen omdat het me een leuk vak leek. Het leek me ook erg interessant omdat alles wat online gebeurt steeds belangrijker word in de wereld en ik er graag meer over zou leren. Daarnaast vind ik het erg interessant hoe technologie de wereld heeft verandert en ik wil daar graag meer van begrijpen. Ik hou ook van de logica achter programmeren en het gevoel van voldoening wanneer mijn code eindelijk werkt. Bovendien biedt informatica veel toekomstmogelijkheden, wat het voor mij nog interessanter maakt.
